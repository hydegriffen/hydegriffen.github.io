Tweetalyze

Tweetalyze is a web application which enables users to access tweet data through a convenient UI to build queries, generate graphics, and export results without the need to learn how to code or clean data. The tool uses a Heroku application to query the Twitter API and leverages the Plotly library to generate graphics. Tweetalyze is created by me and Adam Gincel.

ThreeDee Valley

ThreeDee Valley is an in-development re-imagining of the joint WaveParadigm (Adam Gincel) - HydeGriffen (me) Unreal Engine 4 project ThreeDee using professional assets to supplement our character and collectible models. In an homage to the N64 3D platformer era, a polygonal character wanders a beautiful valley to defeat enemies and gather colorful items.
